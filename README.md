## Program Summary
- Android sample app that checks Bluetooth adapter status, prompts to enable Bluetooth if disabled, runs discovery, and lists paired devices with names/addresses in a TextView.
- Focused on demonstrating basic Bluetooth adapter access and bonded device enumeration.
